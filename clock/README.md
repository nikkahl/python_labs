## Analog Clock Angle Calculator 

  Реалізація алгоритму для обчислення найменшого кута між стрілками годинника.

##  Опис завдання
Програма розраховує кут між годинною та хвилинною стрілками для заданого часу (години, хвилини, секунди). 

### Основні характеристики:
* **Плавний рух**: враховується зміщення годинної стрілки залежно від хвилин та секунд.
* **Точність**: алгоритм працює з точністю до секунд.
* **Внутрішній кут**: результат завжди повертається як найменша дуга ($0^\circ \le \alpha \le 180^\circ$).

### Технічні стандарти
* **Стиль коду**: PEP 8 (`snake_case`).
* **Документація**: Google Style Docstrings.
* **Тестування**: автоматизоване завантаження тестів із файлу `clock_test_data.json`.
* **Логування**: використання модуля `logging` для звітування про результати.
